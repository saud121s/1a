# مساعد التاجر الذكي (Smart Merchant Assistant)

بوت تيليجرام احترافي مصمم لدعم أصحاب المتاجر الإلكترونية في السعودية. يساعد التاجر في تحسين مبيعاته وتسهيل مهامه اليومية.

## المميزات الرئيسية:
- 📝 **كتابة وصف المنتجات:** كتابة أوصاف جذابة ومقنعة بأسلوب سعودي.
- 🏷️ **اقتراح هاشتاقات:** هاشتاقات ترند لزيادة الوصول في تيك توك وإنستقرام.
- 💬 **خدمة العملاء:** صياغة ردود احترافية ولبقة للعملاء.

## كيف يعمل؟
البوت مدعوم بتقنيات الذكاء الاصطناعي (OpenAI) لتقديم أفضل النتائج التجارية بدقة عالية.

## التقنيات المستخدمة:
- Python
- Aiogram (Telegram Bot API)
- OpenAI API
- Railway (Deployment)
